Recursos de Acessibilidade deste Formulário

Características de Acessibilidade

  • Rótulos (labels) corretamente associados a cada campo
  • Indicação clara dos campos obrigatórios
  • Mensagens de erro específicas para cada campo
  • Descrições adicionais para ajudar no preenchimento
  • Campos agrupados logicamente com fieldset e legend
  • Feedback de validação acessível para leitores de tela
  • Navegação por teclado com ordem lógica
  • Contraste adequado para textos e elementos

Implementações ARIA

  • aria-required - Indica campos obrigatórios
  • aria-describedby - Conecta campos a suas descrições
  • aria-invalid - Aplicado dinamicamente para indicar erros
  • aria-live - Região para anunciar erros de validação

Valide o formulário sem preencher campos obrigatórios ou com formato incorreto para experimentar o comportamento acessível de validação.
